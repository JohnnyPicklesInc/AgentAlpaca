🦙 Talk to your agents from anywhere

Your coding agent is running on the computer at home. You're on your laptop at a café. Agent Alpaca shows you the live terminal and lets you type back — no screen-share, no VNC.

Get started

1. Pair once

Sign in, click Add a bridge, and run the pairing command on your home machine.

2. Wrap your agent

Start it through Alpaca instead of directly:

alpaca -- claude

3. Watch from anywhere

Open this app on any device, pick the session, and you're looking at the same live terminal.

How it works

The alpaca command wraps your agent in a real pseudo-terminal on the home machine and mirrors it — both directions — through a per-session relay. You keep using the terminal locally as normal; remote input is injected right into it. Terminal bytes stream through and are never stored on the server.
